# San Lorenzo Tenochtitlán | Cuna Olmeca

Sitio web dedicado a preservar y difundir la riqueza cultural de San Lorenzo Tenochtitlán, el primer gran centro regional de la cultura Olmeca en Veracruz, México.

## 🏛️ Sobre el Proyecto

Este proyecto es una plataforma interactiva que permite a los usuarios explorar la grandeza de la capital más antigua de Mesoamérica, esculpida en piedra volcánica y rodeada por la selva veracruzana.

### Características Principales

*   **Inicio:** Una introducción a la cultura y la historia del sitio arqueológico.
*   **Historia:** Información detallada sobre la cultura Olmeca y San Lorenzo.
*   **Visita:** Guía e información útil para planear tu visita.
*   **Avisos:** Novedades y actualizaciones importantes del sitio.
*   **Galería Visual y Videoteca:** Integración con Firebase para la visualización y gestión dinámica de imágenes y videos.
*   **Accesibilidad:** Opciones de alto contraste, ajuste de tamaño de texto y soporte de traducción a múltiples idiomas (Español, Inglés, Náhuatl, Zapoteco).
*   **Panel de Administración:** Sistema de autenticación con Firebase para gestionar los contenidos multimedia del sitio.

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** HTML5, CSS3, JavaScript Vanilla.
*   **Backend / Servicios:** Firebase (Authentication, Cloud Storage, Firestore/Realtime Database).

## 🚀 Instalación y Uso Local

Para probar y ejecutar el proyecto en tu entorno local:

1.  Clona este repositorio.
2.  Copia el archivo de ejemplo `firebase-config.example.js` a `firebase-config.js` e ingresa tus credenciales de Firebase.
3.  Inicia un servidor local en la raíz del proyecto. Por ejemplo, si tienes Node.js, puedes usar `serve`:
    ```bash
    npx serve
    ```
    O usando Python:
    ```bash
    python -m http.server 3000
    ```
4.  Abre tu navegador en `http://localhost:3000`.
